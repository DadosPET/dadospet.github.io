class  PlayerSpawn
 This script is responsible for keeping track of the player positions when it is changing between scenes. For example: you are in the Cin, and the you enter a teacher room. The game scene will change from Cin to Room. When you leave the room, you obviously wanna get back in front of the Room door in the Cin. But this is not going to happen, because the player will be teleported for the position he is originally set up in the scene. So, when changing scenes, you save the name of the scene you are leaving.
