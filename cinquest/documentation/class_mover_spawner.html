Detailed Description

This classe is reponsible for spawning objects with the Move script attached to it. It spawns a mover, and them make it moves for a targetWaypoint, point by point according to the path that is made by it's previous waypoints. *See Waypoint class for more info.

How to use it: Put it into a gameobject and fill the objects list with what you want to spawn. Then, you just have to create a path connecting waypoints previous variables and drag the last waypoint, the target, to the targetWaypoint variable. That's where the mover's will go once spawned, following the path you made.

Developed by: Higor
