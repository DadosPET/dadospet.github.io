Detailed Description

It's the pause menu. When user presses 'P', player is frozen and the menu appears. Player is able to select the options with the mouse.

How to use it: This class carries the Pause Canvas and all its children. This script is attached to GameManager. When player presses 'P' the game object of the Canvas is loaded. All the changes in the menu itself must be made in Unity selecting the Canvas prefab (Pause Canvas). DO NOT do anything hardcoded! Functions here are the actions that will be called by each button on the menu. In Unity, select the button and go to OnClick() and select the function you want.
