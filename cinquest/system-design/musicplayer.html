Sistema de Sons

MusicPlayer.cs

Todo o gerenciamento de áudio do jogo é realizado pela classe MusicPlayer. Ela possue a capacidade de tocar faixas de músicas em loop, trocar músicas com um efeito de fade in/out e tocar efeitos sonoros de curta duração. 


public void ChangeSong(string scene, float time);
public void PlayFX(string name);

Cada música e cada efeito sonoro que será utilizado em algum momento durante o jogo deve ser declarado no Inspector do GameObject onde o script está inserido. Eles então poderão ser acessados através de uma tabela hash utilizando uma string como identificação. 


Dictionary sceneSong;
Dictionary soundFX;

Para tocar um som, então, basta executar os métodos da classe MusicPlayer estáticamente de qualquer script do código. 


MusicPlayer.Instance.PlayFX (“BOOM”);
