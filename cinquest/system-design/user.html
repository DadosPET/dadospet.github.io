Informações do Jogador

User.cs

A classe user é responsável por armazenar o estado de um jogador durante uma seção de jogo. Um estado é composto por:


public class User : MonoBehavior{
    public string nick;
    Dictionary quests;
		List items;

		...
}
  • nick: o nome do jogador
  • quests: uma lista de todas as quests que ele possue, tendo elas sido iniciadas, concluídas ou não.
  • items: o inventário do jogador
